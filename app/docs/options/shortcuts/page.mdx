# Changing Shortcuts

Follow these steps to customize your shortcut for toggling the answers panel:

1. **Open Extension Options**: Right-click the Zolve extension icon and select **Options**.
2. **Modify Shortcut Keys**: Locate the two black boxes representing the current shortcut keys. Click on any box to start key binding.
3. **Live Key Binding Update**: As you press keys, the system will display real-time updates.
4. **Save Your Shortcut**: Once done, click the black box again to stop recording and save the new shortcut.

> **Note:** This shortcut allows you to toggle the visibility of the answers box on the exam page efficiently.
