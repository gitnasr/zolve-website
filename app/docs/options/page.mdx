# Extension Options


Zolve's **Options Page** provides various customization settings to enhance your experience.

### Key Features:

- **Customizable Shortcuts**: Toggle the answers panel with a shortcut. The default shortcut is <kbd>Ctrl</kbd> + <kbd>Enter</kbd>, but you can modify it via the options page.
- **Claude Server Configuration**: Set up and customize Claude AI integration.
- **Cloudflare Workers Support**: Configure a custom AI model using Cloudflare Workers.
